# yet-another-ygo-card-database

This is a card database generated using data from yugipedia.com.

While there are already a number of such databases, many don't offer JSON files and/or are missing other data.
This database tries to be as complete as possible, also including censorship data, rarities per set and printings per rarity and set.

Community contributions are welcome.
Some errors might be solved automatically, once a card is re-parsed from the wiki. However sometimes it might not be fixed, or the data is missing completely.
The latter is especially the case for artwork-rarity links which are only present in text form on the wiki. As such they need to be manually managed.


## FAQ

- **Is \[card] censored?**
  
  If the card is listed in the censored.json-file or contains a x.1 index, the card has a different artwork from the OCG.
